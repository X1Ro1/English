What does the website consist of?

From the user's point of view, a website is a structure consisting of many blocks. Each of them stores different content on the site. The html layout language is responsible for creating this structure.
The external representation of the site is developed using special css style sheets, which together with html, create a beautiful appearance

What is HTML?

HTML (HyperText Markup Language) is the standard markup language for creating documents designed to be displayed in a web browser. It is used to create and structure web pages and their content. HTML describes the structure of a web page using elements represented by tags. Tags can contain text, images, links, tables, lists, and other elements.

The main elements of HTML include:

1. <!DOCTYPE html> — document declaration indicating the HTML version.

2. <html> — the root element that encompasses the entire document.

3. <head> — contains metadata about the document, such as the title, links to stylesheets, and meta tags.

4. <title> — sets the title of the web page, displayed in the browser's title bar.

5. <body> — contains the content of the web page that is displayed in the browser.

6. <h1> - <h6> — headers of different levels, from the largest (<h1>) to the smallest (<h6>).

7. <p> — a paragraph of text.

8. <a> — a hyperlink.

9. <img> — an image.

10. <div> and <span> — elements for grouping and styling content.

Block structure of the site

To create a clean and organized website layout, you often use a block structure. This involves dividing your web page into distinct sections or "blocks," each serving a specific purpose (e.g., header, navigation, main content, sidebar, footer):

In the example shown, we see that the block structure in most cases is organized in three parts (header, main, footer).

How is website design built?

Creating a website design involves using CSS (Cascading Style Sheets) to style and lay out web pages. CSS works alongside HTML to define how elements should be displayed, including their colors, sizes, positions, and overall visual appearance.
